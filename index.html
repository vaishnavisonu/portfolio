My Skills

HTML- HyperText Markup Language

It is the standard markup language used for creating and structuring the content of web pages and applications. HTML is the backbone of web development and is used to define the structure and layout of a webpage, including text, images, multimedia elements, and hyperlinks. It uses various tags and attributes to format and organize the content, allowing browsers to interpret and display the page correctly.

CSS-Cascading Style Sheets

It is a style sheet language used to describe the presentation and formatting of HTML or XML documents. CSS allows web developers to control the layout, appearance, and design of web pages, ensuring a consistent and visually appealing look across different devices and screen sizes.
